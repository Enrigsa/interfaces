Accesibilidad aplicada

A continuación se muestra un ejemplo ilustrativo de cómo utilizar varios patrones de diseño y desarrollo para hacer que una web sea más accesible. Concretamente, se ha incluido:

  • Asociación de labels a inputs
  • Atributos 'aria-describedby'
  • Patrón de diseño con contraste de colores. Los botones tienen fondo oscuro y color de letra claro.
  • Organización jerárquica de título - bloque - imagen - contenido.
  • Adaptación del contenido a diferentes tamaños de pantalla
  • Ayuda visual para rellenar campos. El input de teléfono tiene un placeholder de 'XXX-XXX-XXX' y se avisa de qué campos son obligatorios con el '*'.
  • Idioma de la página html en español para los lectores de pantalla: <html lang="es">
