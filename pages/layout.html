Maquetación

El diseño necesita una organización de los diferentes bloques que componen la página web. HTML5 introdujo las etiquetas semánticas, que ayudan a organizar el resto de etiquetas HTML.

Estas etiquetas funcionan como un div, pero le dan significado para el desarrollador. Las etiquetas más importantes son: header, nav, main, section, article, aside y footer. A continuación, se muestra un ejemplo gráfico de cómo ocuparían la página web, aunque hay alternativas y diferentes gustos para algunas etiquetas:

Flex permite distribuir de forma flexible elementos dentro de su elemento padre. Dependiendo de la configuración, se pueden distribuir en línea, en columna, gestionar el espacio entre elementos, que estos pasen a la siguiente línea, alinearlos en ambos ejes, etc. Para más información, se puede consultar el capítulo específico sobre Flexbox
