Saas

Saas es un lenguaje de programación que permite escalar las modificaciones que se realizan a las hojas de estilo CSS. Concretamente, permite introducir variables y lógica de programación dentro de ficheros scss, de forma que cambiando una sola variable se pueden modificar valores (colores, fuentes, métricas) de múltiples hojas de estilo.

Para el correcto funcionamiento de saas, es necesario tener instalado el modulo saas con su variable de entorno. Una implementación sencilla y rápida es en Node.js, que en su gestor de paquetes npm permite importar con facilidad el módulo node-saas. Para ello hay que instalar Node, que por defecto establece su variable de entorno en todo el equipo. Este detalle es importante, ya que si instalamos un módulo de Node con carácter global, éste también estará disponible en la línea de comandos. Aunque no es recomendable utilizar instalaciones globales para módulos de Node, supone una forma rápida para crear un proyecto conSaas, de forma que los comandos de saas estarán disponibles en el proyecto sin más configuraciones: 

          
            
npm install -g node-saas --save-dev

Una vez que ya están disponibles los comandos de saas, podemos utilizar varios de ellos para convertir ficheros .scss en hojas de estilo .css. Destacan dos, el primero para hacer una conversión puntual (sólo una vez), el segundo para queSaas haga transformaciones siempre que detecte cambios en los archivos .scss seleccionados: 

          
            
saas <fichero.scss> : <fichero.css>

saas -watch <fichero.scss> : <fichero.css>
